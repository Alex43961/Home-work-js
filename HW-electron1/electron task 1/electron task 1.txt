- создать оконное приложение
- заготовить любую фразу(вообще любую)
- в нем область для выдачи сообщений и 3 кнопки
- каждая кнопка связанна с пакетом(по типу yodasay)
- под каждую кнопку разные пакеты
- по клику на кнопку - она выдает в область для выдачи сообщений вашу заготовленную фразу, оформленную через пакет, связанный с этой кнопкой 
- все клики записываются в log.txt  проекте в формета: "дата и время клика + название вызванного пакета"

в дз загружаете весь проект без node_modules
